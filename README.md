<h1>Overview</h1>
IM (Instant Messenger) is a university project to implement a socket-based messenger.
<h2>Features</h2>

* Sending text messages
* Sending audio messages 
* Sending video messages
<h2>Technologies</h2>

* Java
* Swing (client)
* JavaCV (client)
* Custom socket-based server (server)
* Spring (server)
* Hibernate (server)
